# Week 1 - SIT Starter Pack: Software Development

## ขั้นตอนที่ต้องทำ

### 1. Fork repo นี้ไปเป็นของตัวเอง
- กดปุ่ม **Fork** ที่มุมบนขวาของหน้านี้บน GitHub
- จะได้ repo สำเนาไว้ใน GitHub ของตัวเอง

### 2. Clone repo ลงเครื่อง
```bash
git clone <url-repo-ของตัวเอง>
```

### 3. แก้ไขไฟล์ `my-introduction.md`
เปิดไฟล์ `my-introduction.md` แล้วเขียนแนะนำตัว:

- ชื่อเล่น
- งานอดิเรก
- อาหารที่ชอบ
- สิ่งที่อยากเรียนรู้
- หรืออะไรก็ได้ที่อยากให้พี่ๆรู้จัก

### 4. Commit และ Push
```bash
git add my-introduction.md
git commit -m "แนะนำตัว"
git push origin main
```

### 5. สร้าง Pull Request
กลับไปที่ GitHub → กด **Pull Request** → **New Pull Request** → ส่งมาให้พี่ๆ review
